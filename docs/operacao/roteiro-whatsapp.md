# Roteiro WhatsApp

## Prefills do site (por botão)

### Contato / projeto (foco principal)
Botões: navbar, hero, CTAs mid/final, floating.

```
Olá! Vim pelo site da Marca Digital e gostaria de falar sobre um site / projeto.

Meu nome:
```

Abordagem: descobrir se é site novo ou reforma, indicar plano, propor fechamento.

### Diagnóstico (secundário — quem já tem site)
Botão: seção Problema.

```
Olá! Já tenho um site e gostaria de um diagnóstico gratuito (PageSpeed).

Meu nome:
Link do site:

(Para a análise PageSpeed, o link do site já é suficiente.)
```

### Planos (“Quero este plano”)

```
Olá! Vi o plano Landing no site e gostaria de saber mais.

Meu nome:
```

### Sistemas / E-commerce

```
Olá! Gostaria de pedir orçamento para Sistema / E-commerce.

Meu nome:
```

---

## Depois que a mensagem chega

### Projeto / plano

```
Oi, [Nome]! Vi que você quer falar sobre [site / plano X].

Me conta: é site do zero ou você já tem um hoje? Assim indico o melhor caminho (e, se já tiver site, posso oferecer o diagnóstico gratuito).
```

### Diagnóstico — confirmação

```
Oi, [Nome]! Recebi o link. Vou analisar e te mando o PDF em até 24 horas.

Lembrando: só apontamentos, sem correções. Depois montamos a proposta do site/plano.
```

### Sistemas

```
Oi, [Nome]! Sobre o orçamento de Sistema / E-commerce: me conta em poucas linhas o que você precisa e se já tem prazo ou ideia de investimento.
```
