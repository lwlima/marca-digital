# Escopo e ICP

## Cliente ideal (ICP)

- **Tipo:** prestadores de serviço (clínicas, consultórios, oficinas, profissionais liberais, serviços locais B2C/B2B leve).
- **Necessidade:** site rápido, contato claro (WhatsApp) e/ou Perfil da Empresa no Google.
- **Aquisição (fase inicial):** prospecção manual + conversa no WhatsApp (sem tráfego pago obrigatório).
- **Fora do ICP por enquanto:** e-commerce pesado, marketplace, sistemas custom complexos sem briefing (sempre sob orçamento), clientes que só querem “manutenção” em site de terceiros.

## O que fazemos

- Diagnóstico gratuito (PageSpeed + apontamentos em PDF, até 24h).
- Desenvolvimento: Landing, Institucional, Completo (site + Perfil da Empresa).
- Sistemas / E-commerce: **só sob orçamento**.
- Configuração de Analytics (conta do cliente).
- Hospedagem gerenciada + mensalidade (domínio, e-mail, SSL, backup; CDN quando o stack permitir).
- Franquia de horas de manutenção no plano mensal.
- Perfil da Empresa no Google (incluso no Completo; setup avulso só via proposta, se oferecer).

## O que NÃO fazemos

- Anúncios pagos (Google Ads, Meta Ads, etc.).
- Branding / identidade visual / logo (podemos aplicar o que o cliente já tiver).
- Gestão de redes sociais / conteúdo contínuo de posts.
- Manutenção **exclusiva** de site desenvolvido e hospedado por terceiros (sem migrar o projeto para nós).
- Copy longo de marketing ou estratégia editorial.
- Suporte fora da janela do plano (Landing/Institucional: seg–sex; Completo: todos os dias), salvo acordo.

## Regra de site antigo

Diagnóstico no site atual → se fechar, **desenvolvemos (ou reformulamos) o projeto conosco** + mensalidade.  
Não vendemos “só Gerência” em infra de terceiros.
