# Onboarding — checklist do cliente

Enviar **depois** do aceite e pagamento do setup (ou conforme combinado).

## Mensagem modelo

```
Para começarmos, preciso destes itens. Pode enviar por WhatsApp ou e-mail:

OBRIGATÓRIO
• Nome da empresa (como deve aparecer no site)
• Textos principais (ou rascunhos) — Home / Sobre / Serviços / Contato
• Logo em alta (PNG ou SVG) e cores da marca (se tiver)
• Fotos ou vídeos que possam ir no site
• Domínio desejado (ex.: suaempresa.com.br) — se já tiver, acesso ao registrador
• Conta Google (e-mail) para Analytics — ou autorização para criarmos juntos
• WhatsApp comercial que vai no site
• Endereço e horários (se for local físico)

SE PLANO COMPLETO / PERFIL DA EMPRESA
• Acesso ou convite ao Perfil da Empresa no Google (ou CNPJ + endereço para criarmos/orientarmos)
• Fotos do estabelecimento / equipe se quiser usar no Perfil

OPCIONAL
• Referências de sites que gosta
• Materiais de concorrentes (só para alinhamento)

Prazo de entrega começa a contar quando o checklist mínimo estiver completo.
```

## Checklist interno (você)

- [ ] Pagamento setup confirmado  
- [ ] Proposta/contrato aceitos  
- [ ] Materiais mínimos recebidos  
- [ ] Domínio / DNS planejado  
- [ ] Conta Analytics criada/compartilhada  
- [ ] Hospedagem provisionada  
- [ ] Kickoff comunicado ao cliente (prazo estimado)
